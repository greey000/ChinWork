連接你的 GitHub Repo

Token 只會存在你目前這個瀏覽器分頁裡，直接從你的瀏覽器呼叫 GitHub API， 不會經過任何第三方。建議申請 fine-grained token，只勾選這一個 repo， 權限只給 Contents: Read and write。
申請位置：GitHub 右上角頭像 → Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token
